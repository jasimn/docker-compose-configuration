## Three-Tier Application with Node.js, MySQL, and Nginx
A complete three-tier web application using Docker Compose with Node.js backend, MySQL database, and Nginx frontend.

## **Quick Start**
## 1.Clone and Run
# Clone the repository 
```bash
git clone https://github.com/jasimn/docker-compose-configuration.git
cd docker-nodejs-app
```
## 2.Start the application
```bash
docker-compose up -d
```
## 3. Access the Application
- Frontend: http://localhost:<port-number>

- Backend API: http://localhost/api

- MySQL Database: localhost:3306


Backend: Node.js application

Database: MySQL 8.0
